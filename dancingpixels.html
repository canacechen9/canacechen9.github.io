Dancing Pixels
Apr 30 2026
Experiment Work

#Real-time Audio #Motion-reactive Visual #Interactive Design #WebGL #3D Graphics #Shading Programming #GLSL #Matrix Maths #Point Cloud #Organic Patterns #Computational Design

Role: Individual Project

Dancing Pixels is an immersive WebGL experiment that bridges the gap between sound and space. Using a custom GPU-accelerated engine, it renders 10,000 points that react in real-time to both user motion and audio frequency data. By manipulating the perspective, viewers can transition from observing a digital organism to standing inside a reactive cosmic starfield.
