Overview

GPJ Activations Gallery is an immersive experience application developed for Apple Vision Pro that explores how immersive technologies can be used to present and navigate branded experiences. The project combines interaction design, AR/VR development, and 3D spatial design to create an intuitive, gallery-like environment where users can explore past activations in a more engaging and embodied way.

Problem / Purpose

The project addresses a common limitation in how experiential and event-based work is presented. Activations are often documented through images or videos, which flatten spatial and interactive qualities into passive formats. This creates a disconnect between the original experience and how it is later communicated. GPJ Activations Gallery was developed to bridge this gap by translating these projects into a spatial, explorable format, allowing users to move through and engage with them more naturally.

Process

The application is designed as a virtual gallery where each activation is reconstructed as a 3D experience. I focused on spatial layout, scale, and navigation to ensure the environment felt intuitive within a mixed reality context. Interaction design was developed specifically for Vision Pro, considering gesture-based input, gaze, and proximity to guide how users select, enter, and explore each project. Alongside this, I worked on integrating media assets into 3D space, balancing visual clarity with performance constraints.

A key part of the process involved adapting traditional UX thinking to a spatial environment, where hierarchy, navigation, and feedback are no longer confined to a screen. This required iterative prototyping and testing to ensure interactions remained clear and comfortable within a 3D context.

Outcome

The final application transformed how activation work was presented and improved client engagement and understanding. Early demonstrations led to immediate client interest in adopting Apple Vision Pro for their own brand experiences, opening new opportunities for collaboration.

Beyond the output itself, this project was a valuable learning experience for me in independently exploring emerging technologies, developing spatial applications from scratch, and translating experimental ideas into practical tools that deliver real value in a commercial context.
